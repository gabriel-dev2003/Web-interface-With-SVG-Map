
Esse foi a estrutura de uma interface Web que foi criada com o intuito de atender 
os requisitos solicitados no portifólio do 3° semestre no curso de Tecnólogo em Análise e Desenvolvimento de Sistemas.

A interface for gerada com o intuito de intuitíva, minimalista e com funcionalidades complexas.

"Project.Map" foi intitulado o projeto por conta do mapa SVG interativo que foi o foco pricipal do projeto. 
Mas alem do mapa ele conta com um menu com a logo e as opções de menu (Início, Sobre e Contatos);
conta com uma área de login e registro com estruturas de sobreposição das caixas;
uma área que conte o espaço para a descrição do projeto;
um campo de para Testimonial Box;
Uma área de contato com campo contendo um input para Nome, Telefone, Email e a mensagem;
E ao final botões com bootstraps de redes sociais que redireciona para as páginas das redes.

Apesar de uma interface minimalista a interfaconta com suas complexidades. Tendo a verificação de caractéres
no campo de login, registro, e das informações inseridas nos inputs da área de contato.
